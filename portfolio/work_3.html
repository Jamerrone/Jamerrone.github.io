Project 2

School project where we had to design a mobile application for back hernia patients.

Me and my colleagues had to design a mobile application for hernia patients, we came up with an exercise app with some game elements in it. Users can for an example level up and gain points, this will motivate them to continue the use of the app for their own good, also by leveling up users can unlock new exercises, this provides quite a good variety. The application has a profile page, exercise page, planner page and much more. The planner function is quite an important one, users can build and plan their own exercises, this makes the application much more flexible if you compare it to others.
