James Reddit

A better looking Reddit clone using Ruby on Rails and bootstrap.

I started developing this project to see how fast I could learn a new programming language from scratch, in this case Ruby. It was quite hard at first, but in my option for a 30 hour SRP project I did quite a good job. On this online platform people can sign up/in, share links, edit them, delete them, rate the shared links and last but not least comment on the shared links. The system also recognizes when someone shares a photo, if this is the case, the system will show the picture in the place of the link.
